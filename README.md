## Introduction

BudgetBuddy SA is a personal budgeting application designed to help users manage their finances effectively. The app enables users to record expenses, categorise spending, set financial goals, track progress and improve budgeting habits through gamification features.

---

## Purpose

The purpose of BudgetBuddy SA is to provide users with a simple and user-friendly platform for managing personal finances. The application helps users monitor spending behaviour and achieve financial goals.

---

## Features

### Core Features

* User Login
* Expense Tracking
* Category Management
* Budget Goal Setting
* Progress Tracking
* Graphical Reports

### Advanced Features

* Automatic Budget Progress Monitoring
* Spending Analytics Graphs
* Receipt Image Upload
* Smart Expense Reminders

### Gamification Features
